发表于 2019-01-02 | 分类于 工具 |
字数统计: 373 | 阅读时长 ≈ 1

anaconda是python发行的包的管理工具，其中自带python的版本，还带很多python的包.安装它比起安装python可以省掉再安装python包的时间。并且利用 Python 进行科学计算，就需要一一安装所需的模块， 而这些模块可能又依赖于其它的软件包或库，安装和使用起来相对麻烦但是用Anaconda只需直接输入conda install <包名>，所有依赖的包和库会自动安装好。
