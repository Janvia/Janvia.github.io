名词化 の vs こと

「こと」しか使えない場合

「の」しか使えない場合

たい　VS　ほしい

Ｖたい“想做某事”

Nがほしい”想要某物”

Vてほしい“希望别人做某事”

あいだ　VS　あいだに

あいだに<时点>

あいだ<时段>

Vたら～た<契机~发现>

と

～と<条件> “一～就～、（如果）~就~

～と<反复、习惯>“一~就~ ”

～ないと～ない＜否定性条件＞

よに

Vる / Ｖないようにする

Vる / Ｖないようにしてください
